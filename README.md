## Overview

This repository contains a simple Android integration of the [FaceOnLive SDK](https://faceonlive.com/) for face liveness detection.

The project is written in **Java** and **Kotlin** and can be used as a reference for adding liveness detection to Android applications.

---

## Features

- Face Liveness Detection
- Passive Liveness
- Active Liveness
- Android SDK Integration
- Java Example
- Kotlin Example

---

## Requirements

- Android Studio
- Java
- Kotlin
- Android SDK

---

## About

This is an integration sample built with the FaceOnLive SDK. The code is intentionally kept simple so developers can quickly understand the basic setup and integration flow.

---

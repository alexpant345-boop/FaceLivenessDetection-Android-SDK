## Overview

This repository contains a simple Android integration of the **FaceOnLive SDK** for face liveness detection.

The project is written in **Java** and **Kotlin** and is intended as a lightweight reference for developers integrating face liveness into Android applications.

---

## Features

- Face Liveness Detection
- Android SDK Integration
- Java Sample
- Kotlin Sample
- Simple Integration Example

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/FaceLivenessDetection-Android-SDK.git
```

Then:

1. Add the required SDK library to your project.
2. Configure your SDK credentials.
3. Build and run the application.

---

## Note

The SDK binary files (`.aar` / `.jar`) are not included in this repository.

This project is shared as a simple integration example and can be adapted to your own Android application.
